# UltimateGamepad
Imagine the most amazing gamepad ever, now make it happen!


# Parts
* Teensy 3.1 or 3.2 or Arduino/Genuino Zero
* Adafruit BLE nrf8001
* GY-85 9DOF IMU
* Infrared receiver
* Infrared led
* 3.7v Lipo Battery
* Adafruit Lipoly/Lion charger backpack
* Sparkfun Mux/Demux
* Wireless charger (optional)
